## **`Objetivo`**

`Un tachito que detecte mediante un sensor ultrasónico externo la presencia de una persona, y que abra el tacho durante unos segundos y lo vuelva a cerrar. Por otro lado, un sensor ultrasónico interno que detecte si el tacho está lleno, y que, si es ese el caso, prenda un led rojo. Mientras el tacho no esté lleno y el sensor interior no se active, se mantendrá prendido un led verde.`
